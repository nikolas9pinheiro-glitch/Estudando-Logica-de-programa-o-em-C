# Algoritmos e Lógica de Programação em C

Este repositório contém uma coleção de exercícios e algoritmos desenvolvidos em linguagem C/C++ durante meus estudos de Tecnologia em Banco de Dados na FATEC Bauru. O objetivo é demonstrar o domínio de fundamentos de programação, manipulação de tipos de dados e lógica algorítmica.

## 🚀 Tecnologias Utilizadas
* Linguagem C e C++
* IDE: Visual Studio Code / Dev-C++
* Compilador: GCC

## 📂 Estrutura do Projeto

O repositório está dividido por categorias de complexidade:

### 1. Fundamentos e Entrada de Dados
Arquivos focados na manipulação de variáveis, leitura de teclado (`scanf`, `fgets`) e formatação de saída.
* **Formulario de exibição de dados.cpp**: Cadastro de funcionários com formatação visual de tabelas.
* **Criando variáveis para armazenar.c**: Exemplos práticos de tipos primitivos (`int`, `float`, `char`).
* **Calculo de Medias.cpp**: Algoritmo para cálculo de média aritmética com estrutura condicional (`if/else`) para aprovação.

### 2. Algoritmos Recursivos
Implementações que utilizam a técnica de recursividade para resolução de problemas matemáticos.
* **Estrutura de Recursividade.c**: Cálculo de fatorial de um número fornecido pelo usuário.
* **Exercícios em Fatorial.c**: Algoritmo para soma recursiva de 1 até N.

### 3. Estruturas de Dados
* **PILHA (STACK) - Exercicio.c**: Implementação básica de uma estrutura de dados do tipo Pilha (LIFO), demonstrando as operações de `Push` (inserir) e `Pop` (remover).
